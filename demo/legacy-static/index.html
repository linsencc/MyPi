一览：在叫什么、做什么、是否在走、下一次醒来、上一次结果。

编辑画面单元

每个画面各自作息；以后若加定时规则，也会收进这一页。

随类型变化的选项会自己长出来，新增一种画面不必再造新页面。

每一次刷新留下的脚印：从何时开始、是否顺利、若错了错在哪、画存于何处。

    与真实屏幕相处时，请记住

    • 输出尺寸须匹配屏幕分辨率
    • 颜色模式须匹配屏幕（黑白 / 灰阶）
    • 全刷 / 局刷策略，减少残影
    • 下发前校验格式与尺寸，不合法须明确报错
    • 相同内容建议去重，避免无意义刷屏
