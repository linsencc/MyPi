## 微雪文档（树莓派相关）沉淀

- 来源：Waveshare `13.3inch e-Paper HAT+ (E) Manual`
- 链接：https://www.waveshare.net/wiki/13.3inch_e-Paper_HAT+_(E)_Manual#.E5.85.B3.E9.97.ADSPI.E6.8E.A5.E5.8F.A3

### 文档原步骤摘录（保持原意）

- 进入配置：`sudo raspi-config`
- 菜单路径：`Interfacing Options -> SPI -> No`（文档描述为“关闭 SPI 接口”）
- 重启：`sudo reboot`
- 编辑配置文件：`sudo nano /boot/config.txt` 或 `sudo nano /boot/firmware/config.txt`
- 文档建议追加：
  - `gpio=7=op,dl`
  - `gpio=8=op,dl`
- 保存退出后再次重启：`sudo reboot`

### 当前执行策略（项目决议）

- 鉴于厂商文档步骤与常见实践存在理解差异，`SPI` 与 `I2C` 先保留意见。
- 现阶段不对 `SPI`、`I2C` 做任何改动，维持系统现状。
- 在拿到厂商进一步确认前，不执行：
  - `raspi-config` 中的 SPI 开/关切换
  - `i2c-tools` 安装及 `i2cdetect` 扫描
  - `config.txt` 中与本屏相关的新增 GPIO 条目
