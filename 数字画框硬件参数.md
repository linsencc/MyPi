# 数字画框硬件参数

## 1. 硬件清单

| 项目 | 说明 |
|------|------|
| 显示屏 | Waveshare 13.3″ e-Paper **E6** 全彩电子墨水，**1600×1200**，**SPI** 接口 |
| 主控 | Raspberry Pi Zero 2 W |
| 电源 | PiSugar 3 |

---

## 2. 网络与账户

| 项目 | 值 |
|------|-----|
| hostname | `raspberrypi` |
| 登录用户 | `linsen` |
| 无线 IPv4（`wlan0`） | `192.168.1.118/24` |
| 默认网关 | `192.168.1.1` |
| 默认路由 | `default via 192.168.1.1 dev wlan0` |

---

## 3. 系统环境

| 项目 | 值 |
|------|-----|
| 内核 | `Linux raspberrypi 6.12.47+rpt-rpi-v8` |
| 架构 | `aarch64`（64 位） |
| 发行版 | Debian GNU/Linux **13 (trixie)**，`VERSION_CODENAME=trixie` |
